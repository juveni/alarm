tizen.alarm.add()

API used to add alarms based on relative/absolute time to the storage.

    tizen.AlarmRelative()

    API used to creates an alarm that will occur at fixed interval of time in future.

    • Simple Relative Alarm

      sets an alarm that will trigger after the given time period from the current time

      Demo
      Run Code 
      					// remove all alarm
					tizen.alarm.removeAll();
					
					//Create relative alarm, occurring at a fixed interval in the future.
					var alarm = new tizen.AlarmRelative(2 * tizen.alarm.PERIOD_MINUTE);
					//Add the alarm to the storage.
					tizen.alarm.add(alarm, "org.tizen.clock"); 
					console.log("Alarm added with id: " + alarm.id + "\n" + 
					"It expires after 2 minutes from now");

      The above code sets an alarm that will be triggered after 2 minutes from the current time after it is added.

    • Recurring Relative Alarm

      sets a alarm that will trigger after the given delay duration period from the current time and occurs repeatedly at given intervals afterwards

      Demo
      Run Code 
      					// remove all alarm
					tizen.alarm.removeAll();
					
					var alarm = new tizen.AlarmRelative(tizen.alarm.PERIOD_HOUR,
                                      2 * tizen.alarm.PERIOD_MINUTE);
					tizen.alarm.add(alarm, "org.tizen.clock"); 
					console.log("Alarm added with id: " + alarm.id + "\n" + 
					"It expires after an hour, recurring every 2 minutes afterwards");

      Above code Sets an alarm in one hour, recurring every 2 minutes afterwards.

    • Schedule Application Service

      Launch the given service after specified time.

      Demo
      Run Code 
      				var alarm = new tizen.AlarmRelative(10);
				var service = new tizen.ApplicationService("http://tizen.org/appsvc/operation/view","http://www.tizen.org");
				tizen.alarm.add(alarm, "org.tizen.browser", service);
 				console.log("Alarm added with id: " + alarm.id);

      Above code launch the browser application after 10 sec.

    tizen.AlarmAbsolute()

    API used to create an alarm that will occur at a given date/time.

    • Simple Absolute Alarm

      Sets an alarm that will trigger at a given absolute date/time

      Demo
      Run Code 
      				// remove all alarm
				tizen.alarm.removeAll();
				
				var date = new Date(2013, 0, 1, 8, 0);
				var alarm = new tizen.AlarmAbsolute(date);
				tizen.alarm.add(alarm, "org.tizen.clock");
				console.log("Alarm added with id: " + alarm.id + "\n" + 
					"This alarm expires on January 1st 2013 08:00");

      Above code sets an alarm on January 1st 2013 08:00

    • Recurring Absolute Alarm

      Sets an alarm that will occur repeatedly at given interval or will occur every week on the given array of days starting from the given absolute date.

      Demo
      Run Code 
      				// remove all alarm
				tizen.alarm.removeAll();
					
				var date = new Date(2013, 0, 1, 8, 0);
				var alarm = new tizen.AlarmAbsolute(date, ["SA", "SU"]);
				tizen.alarm.add(alarm, "org.tizen.clock");
				console.log("Alarm added with id: " + alarm.id + "\n" + 
					"This alarm expires on every Saturday and Sunday, at 08:00, starting on January 1st 2013");
